## 6. Кадры и риски

| Риск | Вероятность | Влияние | Митигация |
|---|---|---|---|
| NAPP не отдаёт API в срок | средняя | блокирует MVP | заложить ручной ввод авто как fallback |
| Uzcard / PCI-DSS затянет сертификацию | высокая | задержка релиза | начать собирать документы PCI-DSS параллельно с разработкой |
| Махмуд-ака не успеет передать формулу КАСКО | средняя | КАСКО не войдёт в MVP | релиз только с ОСАГО (всё ещё ценный продукт) |
| Лицензия на страхование не получена к релизу | средняя | нельзя выпускать полисы | закладываем soft-launch с тестовыми пользователями |
| Apple Developer-аккаунт не получен у клиента | низкая | задержка iOS-релиза | зарегистрировать заранее (1-2 недели на проверку) |
| Дизайнер задержит макеты | средняя | сдвиг Этапа 1 | давать макеты по платформам поэтапно (mobile → admin → partner → landing) |
