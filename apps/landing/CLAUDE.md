# apps/landing — sos24.uz (Marketing Site)

> Этот файл дополняет корневой `CLAUDE.md` контекстом, специфичным для лендинга. Корневой читается первым, потом этот.

## Что это

Публичный сайт **sos24.uz**. Цели:
- Объяснить продукт (ОСАГО, КАСКО, комиссары 24/7)
- Привести трафик к скачиванию приложения и оформлению через веб
- Калькулятор стоимости полиса прямо на сайте (с переходом в оформление)
- FAQ, контакты, новости
- SEO для запросов «купить ОСАГО Узбекистан», «КАСКО Ташкент» и т.д.

## Стек

- Next.js 15 (App Router, ISR для контентных страниц)
- Tailwind CSS + кастомные компоненты по бренду
- next-intl (4 локали)
- MDX для статей и FAQ
- Lottie / Framer Motion для анимаций hero
- next/image для оптимизации картинок (фото из `assets/brand/`)
- Аналитика: PostHog + Yandex Metrica (для уз-аудитории привычно)

## Команды

```bash
pnpm dev:landing                       # next dev
pnpm --filter landing run build
pnpm --filter landing run start
pnpm --filter landing run typecheck
```

## Конвенции

- **SEO в первую очередь** — корректные metadata, OG-теги, structured data (JSON-LD)
- **Производительность** — Lighthouse 90+ обязательно, image-оптимизация, ленивая загрузка
- **Бренд:** строго `assets/brand/` (логотип SOS24, цвета `#E61428`/`#010101`)
- **Без хардкода текстов** — всё через next-intl
- **Минимум JS на главной** — статические компоненты по умолчанию, интерактив только где нужно (калькулятор)
