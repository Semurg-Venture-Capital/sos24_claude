Показатели WHOOP — на русском и узбекском

Приложение WHOOP есть только на английском. Мы берём те же данные браслета и показываем их красиво и понятно: восстановление, сон, нагрузка и вся история метрик — с графиками и на родном языке.

Это интерактивный прототип — переключите RU / UZ, чтобы увидеть локализацию. Данные — пример. Экраны строятся на react-native-svg (уже в проекте), без тяжёлых библиотек.

План реализации

Как это построить

Основа интеграции WHOOP уже работает (OAuth, вебхуки, снапшот последних значений). Чтобы появились графики и история — добавляем хранение тайм-серий и пять экранов показателей.

1

Данные и модель

  • Сейчас храним только снапшот последних значений — истории нет.
  • Добавить тайм-серии: WhoopRecoveryDaily, WhoopSleep, WhoopCycle, WhoopWorkout.
  • Бэкфилл истории при подключении (WHOOP отдаёт ~недели назад) + запись по вебхукам.
2

Бэкенд

  • GET /health/wearable/whoop/history — метрика + диапазон (7/14/30/90 дней).
  • Агрегации для трендов (средние, база ВСР, дельты) — на сервере.
  • Переиспользуем BullMQ (бэкфилл + авто-refresh токена) — уже есть.
3

Экраны (мобилка)

  • 5 экранов: Обзор · Восстановление · Сон · Нагрузка · Тренды.
  • Графики на react-native-svg: кольца, гейджи, линии, столбцы сна, зоны пульса.
  • Переход из карточки «Мои показатели» в разделе «Здоровье».
4

Языки и формат

  • Все строки в @sos24/i18n-strings: uz-Latn, uz-Cyrl, ru, en.
  • Числа, даты, единицы — по локали; кДж → ккал, мс, уд/мин.
  • Понятные пояснения к каждой метрике (чего нет в самом WHOOP).
